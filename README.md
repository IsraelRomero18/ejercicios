Encuentra el volumen de un cilindro con radio `5` y altura `10`:

# Cálculo Diferencial e Integral
## Derivadas básicas
1. Derivada de una constante:
d/dx(c) = 0

2. Derivada de `x^n`:
d/dx(x^n) = n * x^(n-1)


## Integrales básicas
1. Integral de una constante:
∫c dx = c * x + C


## Ejemplo
- Derivar `f(x) = 3x^2 + 2x + 1`:
f'(x) = 6x + 2


## Ejercicio
Integra la función `f(x) = 2x`:
- Solución: `∫2x dx = x² + C`
